# Fableous Front-end

## Running the application

Clone repo

```shell script
git clone git@gitlab.com:todo-teamname/fableous.git
# or
git clone https://gitlab.com/todo-teamname/fableous.git

cd fableous/fableous_fe
```

Install dependency

```shell script
npm install
```

Running application in development mode

```shell script
npm start
```
